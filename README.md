## Getting Started

### Setup
```
$ git clone https://github.com/anoopmundathan/smarkets-fe.git
$ cd smarkets-fe
```
### Install
```
$ npm i
```
### Run
``` 
$ npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
